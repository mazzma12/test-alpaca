## Quick Start

Install test_alpaca:

```bash
pip install test-alpaca
```

Display a simple message:

```bash
test_alpaca "Hello World!"
```

Use different alpaca types and colors:

```bash
test_alpaca --alpaca happy --color green "I'm happy!"
test_alpaca --alpaca thinking --color blue "Let me think..."
```
